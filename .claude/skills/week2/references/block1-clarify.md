# Block 1: 방향 명확화 (clarify)

## EXPLAIN

W1에서 여러분은 "나의 선언"을 썼습니다. 그런데 흐린 선언은 AI가 이어받지 못합니다 — *Garbage In, Garbage Out.* 그래서 오늘 첫 손동작은 **내 방향을 선명하게** 만드는 것입니다.

여기서 처음으로 **스킬을 돌려봅니다.** `/clarify`는 team-attention이 만든 스킬로, 열린 질문 대신 **가설을 선택지로** 던져(AskUserQuestion) 모호한 걸 구체적으로 만듭니다. "일단 vague를 갈긴다" — 모호함을 없애는 게 자동화의 시작입니다.

## EXECUTE

Claude Code에 입력하세요:

```
/clarify — journey.md 의 "나의 선언"을 명확하게 만들어줘.
질문에 답하고 나면, 선명해진 버전으로 그 선언을 덮어써 줘.
```

- AI가 3~6개 가설 질문을 줍니다 → 고르기만 하면 됩니다.
- 끝나면 Before/After를 보여주고, journey.md의 `나의 선언`을 **선명해진 버전으로 교체**합니다.

## CHECK

- Before/After에서 내 방향이 더 또렷해졌는지 눈으로 확인한다.
- journey.md의 `나의 선언`이 실제로 갱신됐는지 본다.
- 한 문장 짚어준다: **"방금 여러분은 '스킬을 돌려본' 겁니다. 다음은 직접 하나 만듭니다."**

(시간이 밀리면 이 블록은 건너뛸 수 있다 — Block 2가 오늘의 핵심이다.)
