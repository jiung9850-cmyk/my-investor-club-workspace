# Block 3: 컨텍스트 강화 (ceiling — 준비된 사람만)

## EXPLAIN

Block 2를 끝냈다면 오늘은 이미 성공입니다. 더 갈 사람을 위한 층입니다.

여러분이 부족한 건 도구가 아니라 **좋은 맥락·정보**였습니다. 그래서 다음 단계는 내 하네스에 **진짜 정보를 흘려넣는 연결** — MCP입니다.

핵심 구분: **MCP는 '접근'을, 스킬은 '지침'을 준다.** MCP로 공시·뉴스·노션에 접근해도, 그걸 어떻게 볼지의 규칙(스킬)이 없으면 실패합니다. 둘은 대체재가 아니라 짝입니다. **오늘은 연결하지 않습니다** — 무엇을 연결하면 좋을지 추천만 받습니다(설치는 W3).

## EXECUTE

1. 내 트랙에 맞는 MCP 추천을 받으세요:
```
내 journey.md 의 방향을 읽고, 그 방향에 도움이 될 MCP(외부 도구 연결) 1~2개를
추천해줘. 지금 설치하지는 말고, 왜 그게 내 방향에 맞는지만 설명해줘.
```
   (참고 방향별: 판단→공시(DART)·뉴스 / 커리어→노션·링크드인 / 자동화→증권 API·구글시트)
   추천받은 MCP 이름 1개를 journey.md 에 "다음에 연결할 후보"로 적어 둡니다.

2. (더 갈 사람) 내 트랙의 **반복 작업 하나를 스킬로** 만드세요:
```
내가 매주 반복할 것 같은 {리서치 절차 / 근거 검증 / 멘토링 질문 준비} 를
스킬로 만들어줘. .claude/skills/ 에 저장.
```

## CHECK

- journey.md 에 "다음에 연결할 MCP 후보"가 한 줄 적혔는지 확인.
- (선택) 트랙 스킬이 생겼는지.
- 먼저 끝낸 사람은 옆 사람의 check-in 짓기를 도와주게 한다 — **"정리하고 가르치는 사람이 제일 많이 배웁니다."**
