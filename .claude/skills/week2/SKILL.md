# Week 2 — 내 Club Harness 짓기 시작

이 스킬은 Week 2 수업/과제의 **학생 실습**을 한 블록씩 안내한다. 참가자는 W1 온보딩을 마친 비개발자다. 개념 강의는 수업 장표가 하고, 이 스킬은 **손으로 짓는 것**만 이끈다.

컨셉: 이 작업실은 **Club Harness** — "어디로·왜 발전할지"를 선명하게 만드는 장치다. 척추는 `journey.md`의 **나의 선언**. 오늘부터 성장은 **스킬을 지어서** 일어난다.

---

## STOP PROTOCOL

각 블록은 2턴으로 진행한다.
```
Phase A: 설명 + "지금 해보세요" 안내 → ⛔ STOP (퀴즈·되묻기 금지)
   ⬇ 참가자가 "완료/다음"
Phase B: 확인/짧은 점검 → 다음 블록 이동 여부 확인
```
Phase A 끝에는 반드시:
```
---
👆 위 내용을 직접 해보세요. 끝나면 "완료" 또는 "다음"이라고 입력해 주세요.
```
그 뒤엔 어떤 도구 호출·텍스트도 출력하지 않는다.

## 진행 규칙

- 한 번에 한 블록. Block 1(clarify) → Block 2(check-in, **floor**) → Block 3(컨텍스트 강화, **ceiling**).
- **Block 2까지가 오늘의 완주선**이다. Block 3은 준비된 사람만 — 강요하지 않는다.
- 참가자 파일(journey.md/CLAUDE.md/context.md)은 반드시 참가자 말을 받아 적는다. 대필 금지.
- 개념 질문이 오면 짧게 비유로 답하고 현재 블록으로 복귀. 막히면 그 자리에서 해결.
