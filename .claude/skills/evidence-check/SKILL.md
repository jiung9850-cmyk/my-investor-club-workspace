# Evidence Check — 근거 검증 절차

이 스킬은 "검증 없이는 확정하지 않는 투자자" 선언을 실행하는 반복 절차다. SK하이닉스 ADR 판단과 VC 딜소싱 채널 예습에서 반복됐던 구조를 그대로 스킬로 만든 것이다.

## 절차

1. **대상**: 무엇을 보고 있는지 한 줄로 못박는다 (섹터/종목/질문).
2. **메커니즘 질문**: "A인가 B인가" 형태로, 반증 조건까지 함께 적는다 — 무엇이 확인되면 판단이 바뀌는가.
3. **소스 이력**: 찾은 소스마다 `[1차]`(공시·원데이터·직접 인터뷰) / `[2차]`(언론·리포트) / `[추정]`(AI 추론·회사 전망) 라벨을 붙인다. 벤더·이해관계자가 쓴 소스는 편향을 따로 표시한다.
4. **결정 규칙**: "___ 를 확인하기 전에는 ___로 판단하지 않는다" 형태로 못박는다.
5. **AI 금지 가정**: 참가자 본인의 확증편향 패턴을 겨냥해 적는다 (일반론 아님 — "나는 왜 이걸 믿고 싶어하는가").
6. **Unknowns**: 확인 못 한 것을 목록으로 남긴다. 모르면 수치를 지어내지 않고 `unknown`으로 적는다.
7. **다음 소스 액션**: unknowns를 줄이기 위해 다음에 찾아볼 소스를 구체적으로 적는다.
8. **(선택) 독립 검증**: 반영을 확정하기 전에, 별도 서브에이전트나 타인에게 이 판단을 검증받는다 — 선언의 "매주 판단에 독립 검증을 거친다"를 실제로 실행하는 단계.

## 규칙

- 투자 조언(매수·매도·목표가) 생성 금지.
- 라벨을 붙일 수 없는 수치는 산출물에 쓰지 않는다.
- 참가자가 결론을 요구하면, 먼저 이 절차(특히 메커니즘 질문과 소스 계획)가 채워졌는지부터 점검한다.
