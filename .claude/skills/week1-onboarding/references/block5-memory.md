# Block 5: Memory — 나의 CLAUDE.md 채우기

## EXPLAIN

**수업 복습**: "이번 주에 해야 하는 건 자동화 만들기가 아니라, 내가 어떤 기준을 가진 사람인지 입력하고 저장해서 길러가는 것입니다."

| 원리 | AI가 **매 세션 시작 시 자동으로 가장 먼저 읽는 지시문** — 휘발되는 기억을 파일로 영구화 |
|---|---|
| 비유 | 새로 온 주니어에게 매번 말로 시키는 대신 건네주는 온보딩 문서 |

CLAUDE.md가 채워질수록, 앞으로 이 작업실의 AI는 "나에게 맞는" 답을 한다. 지금 그 첫 페이지를 함께 쓴다. 완벽할 필요 없다 — 매주 조금씩 자란다.

## EXECUTE (인터뷰 — AI가 물어서 채운다)

CLAUDE.md의 `## 소유자` 섹션을 인터뷰로 채운다. AskUserQuestion으로 **하나씩** 묻고, 참가자의 답을 그대로 받아 적는다 (대필·미화 금지).

1. **호칭**: 뭐라고 부르면 되나요?
2. **응답 취향**: AI 답을 어떻게 받고 싶나요? (표로 정리 / 짧은 결론 먼저 / 길더라도 근거까지)
3. **확인 습관**: AI 산출물을 어디까지 직접 확인하고 싶나요? (전부 / 수치·출처만 / 최종본만)
4. **관심의 씨앗** (열린 질문, 있으면): 요즘 관심 있는 산업·기업·투자 주제가 있나요? 없으면 "아직 탐색 중"으로 적어도 좋다.
5. **나의 한 가지 원칙** (있으면): 투자든 일이든, 스스로 지키려는 기준이 하나라도 있나요? (예: "출처 없는 숫자는 안 믿는다") — 있으면 `## 나의 기준`에 첫 줄로 날짜와 함께 넣는다. 없으면 비워두고 "괜찮습니다, 이건 매주 세 줄 결산에서 한 줄씩 쌓입니다"라고 안내한다.

작성 후 참가자에게 CLAUDE.md를 보여주고 "이게 이제 여러분의 AI가 매번 먼저 읽는 규칙서입니다"라고 확인시킨다.
