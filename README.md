# My Investor Club Workspace

**이 레포 전체가 여러분 것입니다.** 그리고 지금은 거의 비어 있습니다 — 의도된 것입니다. 이 작업실은 **5주에 걸쳐 여러분이 직접 짓습니다.** 클럽은 매주 재료(커리큘럼)와 도구(그 주의 스킬)를 공급하고, 방(폴더·절차·스킬)은 수업에서 여러분이 만듭니다.

## 시작하기

이 폴더에서 `claude`를 실행하고:

```
/week1-onboarding
```

## 네 개의 파일 — 이 작업실의 뼈대

| 파일 | 질문 | 성격 |
|---|---|---|
| `curriculum.md` | 클럽이 나에게 무엇을 부어주는가 | **입력** (전원 동일, 클럽이 갱신) |
| `journey.md` | 나는 그 위를 어떻게 지나가고 있는가 | **과정** (나의 궤적·보강) |
| `context.md` | 나는 지금 무엇을 보고 있는가 | **대상** (판단 상태) |
| `CLAUDE.md` | 나는 어떻게 판단하는 사람인가 | **주체** (기준의 침전) |

같은 커리큘럼이 들어가지만, 5주 뒤 이 네 파일은 26명 모두 다르게 자라 있습니다. **그 차이가 이 프로그램의 산출물입니다.**

## 매 세션의 마감 — 세 줄 결산

세션이 끝나면 AI에게 "세 줄 결산하자"라고 하세요:

1. **기준** 한 줄 → CLAUDE.md (오늘 드러난 나의 기준)
2. **판단** 한 줄 → context.md (보고 있는 것에 대한 갱신)
3. **보강** 한 줄 → journey.md (다음까지 채울 것)

## 스킬 — 매주 열립니다

| 시점 | 스킬 |
|---|---|
| 지금 | `/week1-onboarding` (수업 복습 + 기능 탐험 + 자기 선언 = Week 1 과제), `/prep-week2` |
| Week 2 수업에서 | `/week2` + 여러분이 직접 만드는 첫 스킬 |
| 이후 매주 | `/prep-weekN` → `/weekN` … 그리고 여러분의 스킬들 |

강의안·발표안은 받을 때마다 `weeks/week-0N/`(온보딩에서 직접 만드는 서랍)에 **여러분 언어로 정리해** 넣으세요 — 그 순간부터 AI가 읽고 답합니다. 강의 원본이 필요하면(복습·결석) `/club-archive`. 모르는 용어는 그냥 물어보세요: "MCP가 뭐야?"

> 이 레포는 교육 훈련용입니다. 어떤 내용도 투자 조언이 아니며, 모의투자 기록은 실제 매매 판단이 아닙니다.
